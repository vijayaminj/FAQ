FAQ's

What is HTML?

Html stand for Hypertext Markup language.It is a standard text formating language used for developing web pages, html is a language that is interpreted by the browser and it tells the browser how to display and what to display.

What is css?

CSS stands for cascading style sheet, its a style sheet language that determine how the element or content in the page are looked or shown,css used to develop a consistent look and feel for all the pages.

what is javascript?

Javascript is a scripting or programming language that enables you to create dynamically updating content,control multimedia,animate images.
